叉叉网(XX-Net)


主要特性
========

* GoAgent 集成搜索 google ip
* 基于Web 的配置界面、上传部署、Log查看
* 内置了公共 appid, 安装即可上网, 注册自己的google appid
* 新版本提示、更新升级

平台
================
* Windows XP （需要 tcpip.sys 补丁, 比如用 tcp-z）
* Win7
* Win8
* Ubuntu （不能显示系统托盘）
* Debian
